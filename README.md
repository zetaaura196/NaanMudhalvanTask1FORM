# Interactive User Authentication & Assignment Form 🚀

Welcome to my first GitHub web development project! This is a multi-page, responsive web application featuring a modern login screen and an interactive data input dashboard. It was built as part of the **Naan Mudhalvan** skill development task.

🔗 **Live Demo Link:** [View Live Application](https://zetaaura196.github.io/NaanMudhalvanTask1FORM/)

---

## ✨ Features

* **Mind-Pleasing UI Design:** Uses a carefully selected soft, light-neutral color scheme (\(#f4f6f9\)) to minimize eye strain and maximize professional aesthetic appeal.
* **Smart Form Validation:** Fully responsive client-side evaluation rules that verify user scripts and Gmail formats before allowing submission.
* **Interactive DOM Styling:** Includes a live interactive script that alters layout states, highlights required elements dynamically, and allows text styling modification on click.
* **Secured Password View Toggle:** Users can peek or hide their passwords natively using an integrated visibility eye toggle switch.
* **Modular Multi-File Architecture:** Styled components and JavaScript routines are decoupled completely from structural markup pages for enhanced production-ready codebase health.

---

## 📂 Project Architecture

```text
├── index.html          # Main login gate interface wrapper
├── task.html           # Core interactive assignment form layout
├── dashboard.html      # Secure redirect confirmation page
└── css/
    └── style.css       # Unified global structural design layout
```
